#  Fitness Program

## Overview
AI Fitness Program is a web application developed as a university project. It helps users create personalized workout plans based on their fitness goals and personal information. The application also provides progress tracking, secure authentication, and health data processing.

> Note: The current version uses rule-based logic to generate personalized workout plans. It does not use real machine learning models.

## Features

- User registration and login
- Email verification
- Secure password hashing using bcrypt
- Personalized workout plans based on user goals
- Workout progress tracking
- PDF health data extraction
- Progress charts
- Downloadable PDF reports
- Responsive user interface

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/fitness-program.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
